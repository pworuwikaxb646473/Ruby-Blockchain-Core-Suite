# Ruby Blockchain Core Suite 🚀
A powerful, lightweight, and fully decentralized blockchain toolkit built with Ruby.
This project implements core blockchain technologies, consensus mechanisms, cryptography, and Web3 infrastructure for learning, prototyping, and production-ready blockchain development.

---

## 🔹 Project Overview
This repository contains 12 independent, non-repetitive, production-grade Ruby modules for blockchain development.
All code follows modern Web3 standards, supports POW/POS consensus, cryptography, transactions, smart contracts, and decentralized storage.

---

## 📁 12 Core Blockchain Modules
1. **blockchain_core_miner.rb** - POW workload proof mining core, block generation, and blockchain integrity verification
2. **crypto_wallet_generator.rb** - Secure blockchain wallet generator with ECDSA key pair and public address generation
3. **decentralized_transaction.rb** - Decentralized transaction model with signature verification and UTXO structure support
4. **merkle_tree_builder.rb** - Merkle tree implementation for efficient blockchain transaction verification
5. **consensus_pos_validator.rb** - POS proof-of-stake consensus mechanism with staking and block producer election
6. **ipfs_content_hash.rb** - Decentralized storage hash generator simulating IPFS CID content addressing
7. **smart_contract_vm.rb** - Lightweight smart contract virtual machine supporting transfer, lock, and query logic
8. **block_verifier_light.rb** - Light node block validator without full chain synchronization
9. **token_erc20_simplified.rb** - Simplified ERC20 token standard for blockchain token issuance and transfer
10. **p2p_message_protocol.rb** - Blockchain P2P network communication protocol for node data synchronization
11. **chain_synchronizer.rb** - Cross-node blockchain synchronizer supporting longest valid chain selection
12. **zero_knowledge_proof_sim.rb** - Zero-knowledge proof simulator for blockchain privacy computing
